Installation

Using amazon Light Sail, create an instance. This gives you an IP address for a website that you will soon deploy. Then go to www.wordpress.org to download wordpress to your computer. After that, gp back to Light Sail Instance and use "Connect through SSH" option to access MariaDB.

Enter the code below. This will give you a password to access mySQL

cat bitnami_application_password

Use the command below to access the MySQL and it will request the generated password you were given to get access.

mysql -u root -p

Granting SQL Access

Now that we are in MySQL we need to create a database and its user. This will be beneficial into the manual installation of wordpress. Use the following command to create the Database and end it with a semicolon.

CREATE DATABASE wordpress;

Run the next command to create a user and password. And this will also create access for wordpress to your IP address.

grant all privileges on wordpress.* TO 'wordpressdbuser'@'localhost' identified by 'wordpressdbpass1234';

Access Key

You will need a key to access files via cyberduck as you progress through this process. To create the key go to Account (by search bar). From the drop menu click account. From there click on the tab that says SSH Keys.

From here you click the + create button. Choose your zone. Then, if asked, label the key. This will give a key on the instance and a downlaoded one (which you need for Cyberduck).

Cyberduck

Cyberduck is an application that allows you to upload files. After downloading (if needed) open it and open the connection. The shortcut for MacOS is COMMAND + O . Once in the open connection you will need to change the file transfer to SFTP. The password will be generated using a key you created from Lightsail. Once these files are uploaded and chosen you now can edit on WordPress!
