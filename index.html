Create Instance and Download WordPress Using amazon Light Sail, create an instance which gives you an IP address for a website. Then go to www.wordpress.org to download wordpress to your computer. Connect through SSH via Lightsail to access MariaDB. Enter cat bitnami_application_password This gives you a password to access the MySQL. Use the command below to access the MySQL and it will request the generated password you were given to get access. mysql -u root -p Granting Access Now that we are in MySQL we need to create a database and its user. This will be beneficial into the manual installation of wordpress. Use the following command to create the Database and end it with a semicolon. CREATE DATABASE wordpress; Run the next command to create a user and password. And this will also create access for wordpress to your IP address. grant all privileges on wordpress.* TO 'wordpressdbuser'@'localhost' identified by 'wordpressdbpass1234'; Cyberduck This application allows you to upload files. Open it and open the connection. Change the file transfer to SFTP. The password will be generated using a key from Lightsail. Once these files are uploaded and chosen you now can edit on WordPress!
